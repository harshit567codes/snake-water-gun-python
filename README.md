# Snake Water Gun Game (Python)

A simple terminal-based Snake Water Gun game built using Python.

The game allows the player to compete against the computer, which randomly selects Snake, Water, or Gun. The winner is determined according to the game rules.

## Features

- Terminal-based gameplay
- Random computer choices
- Win, lose, and draw detection
- Beginner-friendly Python project

## Technologies Used

- Python 
- random module
